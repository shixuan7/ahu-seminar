### 分享规则
1. 每周三上午8:00-11:00进行组会报告
2. 每人每次分享1篇论文，控制时间在20-30分钟
3. 分享内容包括但不限于：
   - 论文的核心创新点
   - 方法的技术细节
   - 实验结果分析
   - 个人见解与思考
4. 鼓励积极讨论，提出问题和建议
